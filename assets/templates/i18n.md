## Contributing

To add or update a translation, edit the files under `assets/bundles/` (app UI) and `assets/docs/` (README content). The CI regenerates `docs/i18n.md` and all READMEs automatically when changes are detected — no special commit message needed.

1. Copy `assets/bundles/en.json` → `assets/bundles/{code}.json` and translate the values
2. Copy `assets/docs/en.json` → `assets/docs/{code}.json` and translate the values
3. Submit a Pull Request
